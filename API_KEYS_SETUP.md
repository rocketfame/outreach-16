# Налаштування API ключів

## ✅ Централізована конфігурація

**Всі API ключі налаштовуються тільки в одному місці!**

Вся конфігурація API ключів знаходиться в одному файлі: `lib/config.ts`. Це означає, що вам потрібно додати ключі **тільки один раз** в `.env.local`, і вони будуть автоматично використовуватися в усіх частинах додатку.

## Проблема: "Failed to generate topics"

Якщо ви бачите помилку "Failed to generate topics", це означає, що відсутні або неправильно налаштовані API ключі.

## Рішення

### 1. Створіть файл `.env.local` в корені проекту

В кореневій папці проекту (там, де знаходиться `package.json`) створіть файл `.env.local` з наступним вмістом:

```bash
# OpenAI API Key (обов'язково)
# Отримайте ключ на https://platform.openai.com/api-keys
OPENAI_API_KEY=YOUR_OPENAI_API_KEY_HERE

# Tavily Search API Key (обов'язково для генерації тем)
# Отримайте ключ на https://tavily.com/
# Або використайте безкоштовний dev ключ: tvly-dev-...
TAVILY_API_KEY=YOUR_TAVILY_API_KEY_HERE
```

### 2. Де отримати API ключі?

#### OpenAI API Key:
1. Перейдіть на https://platform.openai.com/api-keys
2. Увійдіть або зареєструйтеся
3. Натисніть "Create new secret key"
4. Скопіюйте ключ (він починається з `sk-`)
5. Вставте в `.env.local` замість `sk-ваш-ключ-тут`

#### Tavily API Key:
1. Перейдіть на https://tavily.com/
2. Зареєструйтеся або увійдіть
3. Перейдіть в Dashboard
4. Створіть новий API key
5. Скопіюйте ключ (він починається з `tvly-`)
6. Вставте в `.env.local` замість `tvly-ваш-ключ-тут`

**Альтернатива:** Можна використати безкоштовний dev ключ від Tavily (починається з `tvly-dev-`), але він має обмеження.

### 3. Перезапустіть сервер розробки

Після створення або оновлення `.env.local`:

1. Зупиніть сервер (якщо він запущений) - натисніть `Ctrl+C` в терміналі
2. Запустіть знову:
   ```bash
   npm run dev
   ```

### 4. Перевірка

Після перезапуску спробуйте згенерувати теми знову. Якщо все налаштовано правильно, помилка зникне.

## Важливо!

- **НЕ комітьте** файл `.env.local` в git - він містить секретні ключі
- Файл `.env.local` вже додано в `.gitignore`
- Ключі OpenAI завжди починаються з `sk-`
- Ключі Tavily завжди починаються з `tvly-` або `tvly-dev-`

## Приклад правильного `.env.local`:

```bash
OPENAI_API_KEY=sk-proj-abc123def456ghi789jkl012mno345pqr678stu901vwx234yz
TAVILY_API_KEY=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
```

**НЕ використовуйте ці приклади ключів - вони не працюють!** Вставте свої реальні ключі.
